# Licensing & Support

FHIR4DS is a commercial-grade open-source project. To ensure both the sustainability of the project and the flexibility required by enterprise users, we utilize a dual-licensing model.

## Open Source License

FHIR4DS is licensed under the **GNU Affero General Public License v3.0 (AGPL-3.0)**. 

- **Best for**: Open-source projects, research, and evaluation.
- **Requirement**: If you modify the software or run it as a network service, you must make your source code available under the same license.

## Commercial License

For organizations that cannot meet the requirements of the AGPL-3.0, or for those embedding FHIR4DS into proprietary products, we offer a **Commercial License**.

- **Best for**: Proprietary applications, closed-source enterprise deployments, and high-performance environments requiring priority support.
- **Benefits**: Removes AGPL-3.0 copyleft obligations, provides warranty/indemnity, and includes direct access to the core development team.

## Support & Inquiries

Whether you have questions about the dual-licensing model or need professional support for a production deployment, we are here to help.
